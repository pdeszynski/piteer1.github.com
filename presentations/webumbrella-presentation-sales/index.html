What's a malware?

Malware, short for malicious software, is any software used to disrupt computer operation, gather sensitive information, or gain access to private computer systems.

Every malware has at least 2 basic elements

  • Infection point
  • Command & Control
Before going further - let's look a little bit into the malware. Each malware has at least 3 basic elements ... An infection point as a place where an user gets infected with a malware. It can be an email, a website or a link on a social network Command and control allows Cybercriminal to send updates or commands to an infected user. It can be for example a code which will render a widget with ad on a page or some kind of code which will redirect user out this page.
